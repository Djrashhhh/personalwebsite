My Projects

Centralized Task Tracking & Inventory System

Developed a full-stack inventory and operations management system for a transport management service. Utilized Spring Boot Java for backend development, React.js for frontend, and PostgreSQL for data storage. Implemented user authentication and role-based access control. This system enhanced operational efficiency by enabling real-time tracking, improved issue resolution, and provided valuable data for informed decision-making.

Product Filter Website

JavaScript-based product filtering application with features such as category filtering, name search, and price range selection. Designed to improve inventory management for small trading businesses.

PostgreSQL Appointment-Provider

Developed an interactive appointment tracker using PostgreSQL for a beauty salon. Users select a service, provide their name and phone number, and choose an appointment time. If the phone number exists in the database, the user only confirms the time, as their name is pre-filled. The system outputs a confirmation message with the appointment details.
